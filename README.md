###Command Maps:

##### Lines:
- Tab:
  - `Tab`
- Undo Tab:
  - `ctrl + d`
- Create lines
  - Att top:
    - `ctrl + m`
  - Att bottom:
    - `ctrl + n`
- move lines:
  - up:
    - `alt + arrow up`
  - down:
    - `alt + arrow down`
- cut lines:
  - `ctrl + x`
- copy:
  - `ctrl + c`
- paste:
  - `ctrl + v`
  
##### Undo and Redo:
- Undo:
  - `ctrl + z`
- Redo:
  - `ctrl + shift + z`
